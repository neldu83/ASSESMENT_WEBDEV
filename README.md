# Assessment 2: Big Planetarium - Web Dev Project

## Project Description 
This project is a multi-page website about the solar system.
Each page represents a different planet or moon and includes images and basic structure using HTM, CSS, and Javascript

## Technologies Used 

HTML (structure)
CSS (styling)
JavaScript (interactivity - quiz and random facts)

## Project Structure 

- index.html (Home page)
- callisto.html
- earth.html
- ganymede.html
- io.html
- jupiter.html
- mars.html
- mercury.html
- moon.html
- neptune.html
- pluto.html

- script.js – contains JavaScript for interactivity  
- style.css – contains styling  
- assets/ – contains all images used in the website


## Page Structure 

Each page follows a consistent structure to ensure clarity and usability across the website.

The structure includes:
- A **header**, which contains the website title and a navigation menu. This allows users to easily move between pages using hyperlinks.
- A **main content section**, where information is organised into different sections using semantic HTML elements such as <section>. This helps structure the content clearly and improves readability.
- A **footer**, which provides basic site information and maintains consistency across all pages.

This consistent layout improves user experience by making the website easy to navigate and understand. It also supports accessibility by using clear structure and semantic HTML elements, which help screen readers interpret the content.


## Styling and Interactivity

The website uses **CSS** to control the visual design of the pages. This includes the dark space-themed background colour, white text, spacing, image sizing, navigation styling, and button design. CSS helps keep the pages consistent and improves readability across the website.

The website also uses **JavaScript** to add interactive features. The “Did You Know” button displays random space facts when clicked, and the Jupiter quiz allows users to answer questions and receive feedback. This makes the website more engaging and helps users interact with the content instead of only reading information.

Using HTML, CSS, and JavaScript together allowed me to create a website that is structured, styled, and interactive.


## Features 

- Navigation between pages using links
- Images displayed from the assets folder
- Interactive “Did You Know” button showing random space facts  
- Jupiter quiz with answer checking using JavaScript
- Two educational videos:
- A video on the homepage explaining the solar system
- A video on the Jupiter page explaining the Jupiter and its moons
- Structured layout using semantic HTML  
- Styled pages using CSS
